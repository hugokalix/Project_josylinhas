Arquitetura Front-End

Ateliê Josylinhas

Proposta de Refatoração: Deixando para trás o código espaguete para abraçar uma estrutura modular, limpa e escalável.

1. O Fim dos Pixels Fixos

  • O Problema: Elementos com `width: 920px` quebravam o layout em dispositivos móveis.
  • A Solução: Implementação de unidades relativas (`max-width`) e Flexbox (`flex-wrap`). O design agora é fluido e se adapta nativamente a qualquer tela.
❌ ANTES (artigo.css antigo)
.container {
  width: 920px; /* Engessado */
}
✅ DEPOIS (artigo.css novo)
.container {
  max-width: 920px;
  display: flex;
  flex-wrap: wrap; /* Fluido */
}

2. Conexão DOM e Programação Defensiva

  • A Ponte (DOM): Utilizamos `getElementById` para conectar a lógica de Back-End (JS) aos elementos visuais (HTML), evitando prints sujos no console.
  • A Trava de Segurança: O bloco `if` garante que as funções de serviço só sejam processadas na rota correta (`Sobre.html`). Isso previne vazamentos de memória e erros fatais no console em páginas que não possuem esse painel.
✅ Segurança no script.js
// 1. Mapeamento das saídas visuais na tela
const painelArray = document.getElementById("resultado-array");
const painelSwitch = document.getElementById("resultado-switch");
const painelObjeto = document.getElementById("resultado-objeto");

// 2. Programação Defensiva: Validação de Existência
if (painelArray && painelSwitch && painelObjeto) {
  // A lógica pesada só executa se a tela correta estiver aberta...
}

3. Regras de Negócio na Prática

  • Array (Gestão de Filas): `unshift` e `shift` não são meros comandos; eles simulam a entrada e saída dinâmica de novos orçamentos no Ateliê.
  • Switch (Roteamento): O motor ideal para categorizar escolhas do cliente de forma rápida (O(1) na maioria dos motores JS), simulando um terminal de autoatendimento.
  • Objetos (Entidades): Transformamos a Joseli em uma entidade orientada a objetos, onde suas características (experiência) são mutáveis através de automação (`+= 1`).
✅ Manipulação de Dados (script.js)
// Gestão de Fila (Array)
let servicos = ["barra de calça", "ajuste"];
servicos.unshift("figurino"); // Adiciona ao topo da fila

// Roteador de Decisão (Switch)
switch (opcaoCliente) {
  case 3: servicoEscolhido = "Confecção sob medida"; break;
}

// Entidade Modelada (Objeto)
let costureira = { nome: "Joseli", experiencia: 5 };
costureira.experiencia += 1; // Automação de incremento anual

Visão Geral da Nova Estrutura

Arquitetura modular das pastas garantindo separação de responsabilidades (SoC).

Escalonamento: Componentização

  • A separação de Assets (CSS/Img/JS) preparou nosso projeto para ser convertido nativamente para React ou Angular.
  • O Header e o Footer, antes repetidos, estão perfeitamente isolados para se tornarem Componentes Virtuais (JSX).
⚛️ atelie-josylinhas-frontend/ (Arquitetura React SPA) 
├── 📁 src/                    
│   ├── 📁 assets/             <-- Nossa pasta 'img' vem pra cá!
│   ├── 📁 components/         
│   │   ├── Header.jsx         (Lógica do global.js entra aqui)
│   │   └── PainelJS.jsx       (O painel com a Trava vira componente isolado)
│   ├── 📁 pages/              
│   │   └── Sobre.jsx          
│   └── 📁 styles/             
│       └── global.css         (O arquivo global que criamos!)

Escalonamento Back-End: APIs REST

  • Usando Node.js e Express, a lógica pesada de arrays e objetos sai do Front-End.
  • O servidor processa dados de forma Assíncrona e Não-bloqueante, devolvendo os orçamentos em formato JSON.
🟢 atelie-josylinhas-backend/ (Arquitetura Node.js) 
├── 📁 src/                    
│   ├── 📁 controllers/        (Onde a lógica do nosso Switch vai rodar)
│   ├── 📁 models/             (Onde o Objeto 'costureira' vira Banco de Dados)
│   ├── 📁 routes/             (URLs de comunicação Front -> Back)
│   └── server.js              (Motor principal)

A Joia da Coroa: Escalando com Django (MVT)

  • O Framework perfeito para escalar negócios. Com a arquitetura base pronta, transformamos arquivos estáticos em Templates Dinâmicos instantaneamente.
  • A integração gera um Painel Administrativo out-of-the-box para a dona do Ateliê.
